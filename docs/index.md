# 我的文件網站

歡迎來到我的 MkDocs 文件網站。

這個網站用來整理我的學習筆記、環境安裝流程，以及常用技術主題。  
目前網站已經部署到 GitHub Pages，可以直接透過網址公開瀏覽。

## 這個網站有什麼內容

### 基本頁面
- 關於：網站用途與簡介
- Python / 環境安裝：安裝與設定流程
- Python / 基本語法：常用語法整理

### 適合誰看
- 想快速找到操作步驟的人
- 想把筆記整理成網站的人
- 想用 MkDocs 建立公開文件站的人

## 建議閱讀順序

1. 先看「關於」，了解網站用途
2. 再看「Python / 環境安裝」
3. 最後看「Python / 基本語法」

## 這個網站怎麼做的

這個網站使用：

- MkDocs：把 Markdown 轉成靜態網站
- Material for MkDocs：提供較完整的文件站外觀
- GitHub Pages：提供公開網址

## 更新方式

之後我只要修改 `docs/` 裡的 Markdown 檔案，重新部署後，網站內容就會更新。
